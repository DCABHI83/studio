# **App Name**: PocketWise

## Core Features:

- Add Expense: Allows users to add new expenses with fields for amount, category, date, and optional note.
- Expense List View: Displays a list of all expenses, sorted by date, showing amount, category, date, and note. Provides the ability to delete an expense.
- Monthly Summary: Show total expenses for the current month with a category-wise breakdown. Powered by tool that provides extra features and analysis
- Data Persistence: Store expenses locally in Firestore database so data persists after page refresh.

## Style Guidelines:

- Primary color: Light green (#A7D1AB), for a fresh and calming feel that implies growth and financial wellbeing.
- Background color: Very light green (#F0FAF2). This barely-there tint provides enough contrast to ensure readability, while harmonizing well with the other colors.
- Accent color: Muted blue (#77B5FE), for interactive elements and highlights. It provides a visual cue that stands out while keeping a clean aesthetic.
- Body and headline font: 'Inter' sans-serif, for a modern, neutral look
- Simple, minimalist icons for categories, navigation, and actions.
- Mobile-first, fully responsive layout with clear visual separation between sections using whitespace and subtle dividers.
- Subtle hover effects and transitions for buttons and interactive elements.
